## AI Journey

> LM → LLM → Foundation Models → AI Engineering

| Term | Description | Example |
|------|-------------|---------|
| **LM** | Predicts text based on learned language patterns. | n-gram model |
| **LLM** | Large-scale LM capable of reasoning, text generation, and complex tasks. | GPT-4 |
| **Foundation Model** | General-purpose AI trained on diverse data (text, images, code) for multiple tasks. | Wu Dao 2.0 |
| **AI Engineering** | Builds real applications using foundation models, including APIs, workflows, and automation. | GPT-4 chatbot |
