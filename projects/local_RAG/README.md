# installation 

- clone the project with: 
- ` git clone https://github.com/zamanrahimi/ai-engineer.git`

## Backend 

* install requiremetns
- `pip -r requirements.txt`

* run FastAPI Server 
- `uvicorn main:app --host 0.0.0.0 --port 8001 --reload`
- port can be changed 
- Then take the URL (if port changed) and update App.tsx in frontend/src/app.tsx



## Frontend 

* when you cloned the project then go to frontend directory of the project and run 

- `npm install react-scripts@5.0.1 --save` 
- 5.0.1 might the replaced with latest keyword 
- then start React server with: 
- `npm start`
